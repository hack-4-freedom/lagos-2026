MiraBit
Hack4Freedom Lagos 2026

Overview
MiraBit is a decentralized savings platform designed to help users grow their Bitcoin holdings by converting spare Naira or USDT into a secure, accessible savings pot. Features include automated goal setting, progress tracking, and an integrated learning module to earn sats.

Problem
Many users in Nigeria lack access to inclusive financial tools that protect their savings from inflation while offering a simple, beginner-friendly entry point into Bitcoin.

Solution
MiraBit provides a user-friendly interface to convert local currency (NGN) and stablecoins (USDT) into Bitcoin. It gamifies the experience through an educational "Learn-to-Earn" module, incentivizing users to grow their savings as they gain financial literacy.

Technology stack
Frontend: React, Tailwind CSS, Shadcn UI

State Management: LocalStorage

Blockchain/Bitcoin: Lightning Network integration

API/Data: Real-time Bitcoin/Fiat conversion rates
